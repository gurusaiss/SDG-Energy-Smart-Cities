# EcoCity AI

An AI-powered dashboard for optimizing urban infrastructure, developed for the Devpost Hackathon.

## Features

### 1. Energy Optimization
- **Building Analytics**: Monitor energy usage across commercial and residential sectors.
- **AI Recommendations**: Simulate AI-driven retrofits (e.g., Smart HVAC, IoT Lighting) to calculate potential savings.

### 2. Smart Transport
- **Fleet Management**: Real-time tracking of public transport units.
- **Predictive Scheduling**: AI detects demand spikes and suggests fleet reallocation.

### 3. Renewable Integration
- **Grid Balancing**: Visualize supply vs. demand curves.
- **Load Forecasting**: Manage battery storage and renewable output (Solar/Wind) to maintain grid stability.

## Tech Stack
- **Frontend**: React (Vite), TypeScript
- **Styling**: TailwindCSS (Custom Dark/Eco Theme)
- **Charts**: Recharts
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.
